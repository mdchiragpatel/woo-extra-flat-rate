WooCommerce Extra Flat Rate Shipping plugin is for add new flat rate option in your WooCommerce site.

== Description ==

WooCommerce Extra Flat Rate Shipping plugin is provides you an interface in WooCommerce setting section "Extra Flat Rate". Admin can add Multiple Shipping option or remove any existing shipping.
All Shipping option is display in Front side so User can choose shipping method based on that.

This plugin is for these who wants to use Multiple shipping on there website. By using this plugin, You can add multiple shipping in your website as well as you can remove it as per your requirement.

Please note: This plugin is only compatible for WooCommerce version (2.4.0 and more)

== Installation ==

* Download the plugin
* Upload the folder "woo-extra-flat-rate" to wp-content/plugins (or upload a zip through the Wordpress admin)
* Activate and enjoy!

== Frequently Asked Questions ==

== In which WordPress version this Plugin is compatible?

It is compatible from 3.8 to 4.3.1 WordPress version.

== In which WooCommerce version this Plugin is compatible? ==

It is compatible for 2.4.0 and greater than WooCommerce plugin
